## Features

- Modern floating dock.
- Designed specifically for Cinnamon Desktop.
- Fast startup.
- Application launcher powered by Rofi.
- Pinned applications.
- Running application indicators.
- Window toggle (minimize / restore / focus).
- Network controls.
- Volume control.
- Battery indicator.
- Lightweight.
- Highly configurable.
- Nerd Fonts support.
- Emoji support.
- Modular architecture.
- Fully open source.

## Installation

Clone the repository:

```bash
git clone https://github.com/alons2080-wq/PolyMint.git
cd PolyMint
```

Run the installer:

```bash
chmod +x install.sh
./install.sh
```
